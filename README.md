### Now
Building [**Cred**](https://cred.ninja): OAuth credential delegation middleware for AI agents. When an agent needs access to a user's Google Calendar, Slack, or GitHub, Cred brokers a scoped token. Refresh tokens never leave the vault.

`@credninja/sdk` · `@credninja/oauth` · `@credninja/vault` · `@credninja/mcp` · `@credninja/guard`

7 provider adapters · AES-256-GCM vault · Ed25519 audit trail · MIT licensed

Also building **Obscura**: an authenticated photography registry on testnet.
